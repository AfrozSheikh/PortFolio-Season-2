# Afroz Sheikh's Interactive Terminal Portfolio

Welcome to my interactive portfolio! This project is a unique and engaging way to explore my skills, projects, and experience through a simulated terminal interface and an AI-powered assistant.

## Features

- **Terminal Interface**: A fully functional terminal where you can use commands to learn more about me.
- **AI Assistant**: Chat with my AI assistant to get answers to your questions in a conversational way.
- **Responsive Design**: The portfolio is optimized for both desktop and mobile devices.
- **Multiple Themes**: Switch between `light` and `dark` modes to suit your preference.

## How to Use

### Terminal Mode

When you first open the portfolio, you'll be greeted by the terminal. You can start by typing the `help` command to see a list of all available commands.

Some key commands include:
- `about`: Get a summary about me.
- `skills`: See a list of my technical skills.
- `projects`: Explore the projects I've worked on.
- `experience`: View my professional experience.
- `contact`: Find out how to get in touch with me.
- `theme <dark|light>`: Change the visual theme.
- `clear`: Clear the terminal screen.

### AI Assistant

Click on the **"Assistant"** button to switch to the chat interface. You can ask the AI anything about my profile, such as "Why should I hire you?" or "Tell me more about the SaaS monitoring project."

## Tech Stack

This portfolio is built with modern technologies, including:

- **Next.js**: For a fast and optimized React framework.
- **React**: For building the user interface.
- **TypeScript**: For type-safe code.
- **Tailwind CSS**: For modern and responsive styling.
- **Genkit (Google AI)**: To power the AI assistant.
- **shadcn/ui**: For beautifully designed and accessible UI components.
